TweetGrej
========
Lazy-ass Chrome Extension to send Tweets!

Add to your chrome Extensions.
Config your Twitter.dev app.
Config a server ( see example-server.js for a node.js express server)

Surf the web, find something cool.
Open TweetGrej
Write something.
Save. take a beer.


Open with:
CTRL+B

Send Tweet:
CTRL+ENTER
